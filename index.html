01 · About
A bit about me

I'm a Data Engineer based in Austin, currently building data infrastructure and ML systems for Midtown Athletic Club in Chicago. My day-to-day is mostly Azure Databricks, PySpark, and Python. I write a lot of SQL, deploy XGBoost models, and try to keep pipelines from breaking.

On the side, I write and maintain autolineage, an open-source Python library that automatically tracks data lineage across pandas, NumPy, and scikit-learn pipelines via runtime function hooking. It's on PyPI and GitHub. The goal is reproducibility for ML workflows and a sane path toward EU AI Act Article 10 compliance, without users having to change a line of code.

I have an M.S. in Computer Science from the University of Wisconsin Milwaukee. Before that, I built data pipelines for healthcare at Optum Care and consulted in Delhi. These days I'm spending more time on open source, technical writing, and building with LLMs.
